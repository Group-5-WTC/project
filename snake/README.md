# Snake

In this project, the student implements the classic game of Snake. The game is played on a grid, where the player controls a snake that moves around the grid. The snake grows in length as it eats food, and the game ends when the snake collides with itself or the boundaries of the grid. The student is encouraged to be creative in designing the user interface and the game mechanics.

## Learning outcomes

- User interface design
- State management
- Creative confidence
