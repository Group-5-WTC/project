# Notebook

In this project, students are asked to implement a simple notebook application. The application should be able to create, read, update, and delete notes. The application should be able to handle multiple notes.

It may be similar to Notion or Evernote, and is particularly well suited for web or mobile platforms.

## Learning outcomes

- User interface design
- State management
