# Raytracer

In this project, students develop a minimum viable raytracer. The raytracer is capable of rendering simple scenes with spheres and planes. The raytracer is able to handle shadows, reflections, and refractions.

## Learning outcomes

- Optimization
- Algebra in software engineering
